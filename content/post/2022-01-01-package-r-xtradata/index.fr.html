Ce package R propose d’interroger les webservices xtradata de Bordeaux Métropole. Ces webservices permettent d’interroger les couches de données présentes sur l’open data de Bordeaux Métropole, et également de réaliser des opérations mathématiques sur les données temps réelles historisées.

En l’état seuls les services features et aggregate peuvent être interrogés via ce package.

Davantage d’informations sur ces webservices sont disponibles ici : webservices xtradata BM.

L’utilisation des services (et par conséquent du package) nécessite de posséder une clé d’accès. Demande de clé.
